## Hello world example for WebAssembly in Go

#### Build:

```
make build
```

#### Run:

```
make run
```

#### Clean:

```
make clean
```

### Note:

*wasm_exec.js* and *index.html* in the folder *static* are from [https://github.com/golang/go/tree/master/misc/wasm](https://github.com/golang/go/tree/master/misc/wasm)
